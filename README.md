# Enrollment Management System

## 🛠️ How to Setup
1. **Database:** Open MySQL Workbench, create a new schema named `enrollment_system`, and **Import** the provided `.sql` file.
2. **Libraries:** Ensure the `lib` folder is kept in the same directory as the executable.
3. **Run:** Double-click `EnrollmentSystem.jar` to launch the application.

## 📝 Project Features
- **Student Management:** Full CRUD (Create, Read, Update, Delete) for student records.
- **Course Management:** Add and view available academic courses.
- **Enrollment Logic:** Link students to specific subjects in the database.
